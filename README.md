# IT Homelab

A practical IT lab documenting hands-on work across Linux administration, Windows administration, networking, sysadmin infrastructure, and IT security. Built on an Oracle Cloud VPS and local VMs running in Hyper-V.

## Structure

| Phase | Folder | Status |
|---|---|---|
| PowerUser | [PowerUser/](PowerUser/) | Complete |
| Networking | [networking/](networking/) | In progress |
| Sysadmin | [sysadmin/](sysadmin/) | Upcoming |

## Environment

- **Linux:** Oracle Cloud VPS, Ubuntu 24.04 LTS, accessed over SSH
- **Windows:** Local Windows 10 machine and Hyper-V VMs (Windows 10, Windows Server 2025)
- **Repo policy:** No credentials, keys, IPs, or sensitive system details are committed
