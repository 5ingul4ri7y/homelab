# it-homelab

A 12-week practical IT lab documenting hands-on work across
Linux administration, networking, sysadmin infrastructure,
and IT security. Built on an Oracle Cloud VPS.

## Structure
- phase-1-linux/     — Linux power user skills
- phase-2-networking/ — Networking fundamentals
- phase-3-sysadmin/  — Server configuration & AD
- phase-4-security/  — Security, hardening, incident response
